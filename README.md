# Web-VIEW
# 网页访问器

这是一个简单的网页访问器，只需在输入框输入网址，然后点击“打开网页”按钮，即可在新窗口打开指定网址。

## 使用方法

1. 打开 `web-accessor.html` 文件（用浏览器打开）。
2. 在输入框中输入想要访问的网址，例如：
   - `www.baidu.com`
   - `https://www.example.com`
3. 点击 **打开网页** 按钮，该网址会在新窗口中打开。

> **注意**  
> - 如果没有填写 `http://` 或 `https://`，系统统会自动补全 `https://`。
> - 仅供学习和测试用途。部分浏览器可能会拦截弹出窗口，请允许弹窗访问。
